## 💡 小喵喵的优势

1. **性价比极高** —— DeepSeek 成本低，国内访问友好，支付也方便
2. **保留 Copilot 使用习惯** —— 输入 `@cat` 可直接与小喵喵聊天、辅助开发
3. **快速生成代码** —— 编辑器任意位置输入 `@cat` 即可根据当前文件语言生成对应代码片段
4. **安全可控** —— 小喵喵不会直接修改你的代码，只会以提示形式出现

示例：

```typescript
// 复制字符到剪切板 @cat
```

输入 `@cat`，稍等片刻即可在右侧自动出现生成的代码片段。
